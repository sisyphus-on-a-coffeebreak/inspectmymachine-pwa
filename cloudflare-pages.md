# Cloudflare Pages Deployment

## Setup Instructions

1. **Connect Repository to Cloudflare Pages**
   - Go to [Cloudflare Dashboard](https://dash.cloudflare.com)
   - Navigate to **Pages** â†’ **Create a project**
   - Click **Connect to Git**
   - Select your GitHub repository: `sisyphus-on-a-coffeebreak/inspectmymachine-pwa`
   - Select branch: `main`

2. **Build Configuration**
   - **Framework preset**: Vite
   - **Build command**: `npm run build:production`
   - **Build output directory**: `dist`
   - **Root directory**: `/` (leave empty)

3. **Environment Variables** (if needed)
   Add these in Cloudflare Pages settings:
   - `NODE_ENV=production`
   - `VITE_API_ORIGIN=https://api.inspectmymachine.in/api`
   - `VITE_APP_URL=https://inspectmymachine.in`

4. **Custom Domain**
   - After deployment, go to **Custom domains**
   - Add `inspectmymachine.in`
   - Cloudflare will automatically configure DNS

## Automatic Deployments

Cloudflare Pages will automatically deploy when you push to `main` branch.

## Manual Deployment

You can also trigger deployments manually from the Cloudflare dashboard.
